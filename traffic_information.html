交 通資訊－台南 ‧ 富信大飯店
  • 如何往返［台南高鐵站、台南火車站］
    • 高鐵時刻表與票價查詢  /  台鐵列車時刻查詢系統
    • 高鐵站 → 火車站
    • 台南高鐵站 → 出驗票閘門後左轉 → 步行約 3 分鐘 → 台鐵沙崙站 → 搭乘沙崙線 → 台南火車站
    • 火車站 → 高鐵站
    • 台南火車站 → 至 2 號月台候車 → 搭乘沙崙線 → 台鐵沙崙站 → 步行約 3 分鐘 → 台南高鐵站
    • 搭乘小提醒：
      • 台鐵沙崙站為沙崙線之終點站，班班皆達台南火車站。
      • 沙崙線行車時間約 23 - 25 分鐘。
      • 敬請注意高鐵、台鐵之出發及到達時間，若需購票或取票，請預留 15 分鐘以上時間。
  • 自行前往
    • 飯店地址： 704 台南市北區成功路 336 號
    • 交通方式：
      • 搭乘高鐵
      • 搭乘高鐵至台南站 → 轉乘高鐵接駁車至興南客運台南站下車 → 步行約 20 分鐘即可抵達
      • 搭乘火車
      • 搭乘火車至台南站 → 沿成功路直行 → 步行約 15 分鐘即可抵達
      • 行車路線
      • 永康交流道下 → 往市區方向 → 沿台 1 線直行 → 經西門路 → 右轉成功路即可抵達
      • 台南交流道下 → 沿 182 縣道直行 → 經東門圓環 → 沿府前路直行 → 右轉西門路 → 左轉成功路即可抵達
      • 停車資訊
      • 提供汽車停車位 35 個、機車停車位 30 個，恕不提供保留服務，詳情請洽飯店官網
    • 飯店簡介：
    • 府城古典巴洛克式建築－台南 ‧ 富信大飯店。2014 年全新完工，西門路與成功路口，近台南火車站，臨近國華街小吃及海安路藝術街，著名一級古蹟赤崁樓、大天后宮、孔廟等比鄰而立，步行可至在地老字號美食富盛號碗粿、阿松割包、武廟肉圓等，飯店絕佳的地理位置，是商務洽公、旅遊住宿的首選。
    • 台南富信飯店以巴洛克宮廷式建築風格興建，星級飯店規格打造，提供旅客便利舒適的專業服務。全館共 116 間客房，典雅貴族的裝潢風格，與當地文化特色交融，一樓富麗廳提供自助中西式早餐，接待中心協助安排旅遊路線及景點諮詢服務，健身房、KTV、商務中心等設施，滿足住宿旅客完美驚豔的五感體驗。
